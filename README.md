# ConfMS - Conference Abstract Management System

A complete web-based system for managing conference abstract submissions, peer review, and program scheduling.

## Features

- 🔐 **Multi-role Authentication** - Admin, Author, and Reviewer portals
- 📝 **Abstract Submission** - Multi-step wizard with autosave
- ⭐ **Peer Review System** - Structured scoring rubric (1-10 scale)
- 📊 **Analytics Dashboard** - Real-time submission and review metrics
- 👥 **User Management** - Role-based access control
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile
- 💾 **Local Storage** - Data persistence without backend
